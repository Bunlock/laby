0) installer express : 
> npm install -g express express-generator (g = global)

1) Creer l'architecture du dossier:
> commande express -e (pour avoir ejs)

2) NPM install (va chercher toutes les dépendances listées dans le package.json)

3) installer moongoose (orm de mongo)
npm install --save mongoose (--save permet de l'intégrer dans notre package .json)

4) installer express-session et connect-mongo

5) On controle: npm start
//@client :  localhost:3000
